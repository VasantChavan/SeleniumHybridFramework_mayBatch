step 1: git is installed or not 

git --version if not installed then download git
from https://git-scm.com/

step 2: create a account on git as an remote respository

go to https://help.github.com sign up and then sign in

step 3: go to location of your maven project throgh cmd .

step 4: clone remote repository to that location

git clone "location of your remote repository"

step 5: check out git status 

git status

step 5: do some changes in your maven project. and then check status 
of git repository.

git status it will display whatever changes we did in maven project in red colur.

step 6: git add .  it will add all files and folders.

step 7: git commit -m "message "

step 8:git push origin master
